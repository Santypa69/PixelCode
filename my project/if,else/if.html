Sentencias Condicionales

Esta pagina va a ser extensa porque tocamos todos los temas importantes

Muy a menudo, cuando escribes código, deseas realizar distintas acciones distintas condiciones. Puedes usar declaraciones condicionales en tu código para lograrlo.

En PHP tenemos las siguientes declaraciones condicionales:

  • if Declaración: ejecuta algún código si una condición es verdadera.
  • if...else declaración: ejecuta algún código si una condición es verdadera y otro código si esa condición es falsa.
  • if...elseif...else Declaración: ejecuta códigos diferentes para más de dos condiciones.
  • switch Declaración: selecciona uno de los muchos bloques de código que se ejecutarán.

PHP - La sentencia if

Para comparar dos valores, necesitamos utilizar un operador de comparación.

Estos son los operadores de comparación PHP para usar en if declaraciones:

Tabla de Operadores
Operador Nombre Resultado
== Igual Devuelve verdadero si los valores son iguales
=== Idéntico Devuelve verdadero si los valores y los tipos de datos son idénticos
!= Diferente Devuelve verdadero si los valores son diferentes
<> Diferente Devuelve verdadero si los valores son diferentes
!== No idéntico Devuelve verdadero si los valores o los tipos de datos no son idénticos
> Mayor que Devuelve verdadero si el primer valor es mayor que el segundo valor
< Menor que Devuelve verdadero si el primer valor es menor que el segundo valor
>= Mayor o igual que Devuelve verdadero si el primer valor es mayor o igual que el segundo valor
<= Menor o igual que Devuelve verdadero si el primer valor es menor o igual que el segundo valor

Estos son los operadores lógicos PHP para usar en if declaraciones:

Operador Nombre Descripción
and Y Verdadero si ambas condiciones son verdaderas
&& Y Verdadero si ambas condiciones son verdaderas
or O Verdadero si al menos una de las condiciones es verdadera
|| O Verdadero si al menos una de las condiciones es verdadera
xor XOR Verdadero si una y solo una de las condiciones es verdadera
! NO Verdadero si la condición es falsa

Podemos comparar tantas condiciones como queramos en una sola if afirmación
